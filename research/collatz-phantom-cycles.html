1. Introduction

The Collatz conjecture states that for every positive integer n, the sequence defined by repeatedly applying n → 3n+1 if n is odd, n → n/2 if n is even, eventually reaches 1. Posed by Lothar Collatz in 1937, it remains one of the most accessible yet stubbornly unresolved problems in mathematics. Paul Erdős remarked: “Mathematics is not yet ready for such problems.”

Any proof of the conjecture must rule out two scenarios:

Case A — Cycles
A sequence that loops forever: n0 → n1 → … → n0. The only known cycle is 1→4→2→1. Are there others?
Case B — Divergence
A sequence that grows without bound. No such sequence has been found; all n < 2.7×1020 have been verified to converge.

This document focuses on Case A: the cycle case. We develop a framework based on phantom cycles — periodic orbits of a family of finite modular maps — and show that these exhibit a beautiful dissolution cascade structure that strongly constrains any hypothetical real Collatz cycle.

⚠ Important Notice
The framework presented here does not constitute a complete proof of the Collatz conjecture. The phantom cycle structure is genuine and interesting, and the cascade compatibility theorem is rigorously proved within the phantom cycle world. However, the connection between phantom cycles and genuine Collatz cycle elements requires an additional assumption (the Phantom Cycle Lifting Conjecture, §6) that is not yet proved. The document identifies this gap precisely.

2. The Macro-Step Formulation

Rather than working with the raw Collatz map, we use the macro-step, which jumps directly from one odd number to the next odd number in the orbit, skipping all even intermediates.

Definition — 2-Adic Valuation
For any positive integer x, define v₂(x) as the largest power of 2 dividing x: the unique integer k such that 2k | x but 2k+1 ∤ x.
Definition — Macro-Step
For an odd positive integer n, define:
  • K = v₂(n+1)   — how many times n+1 is divisible by 2
  • m = (n+1) / 2K   — the odd part of n+1
  • x = m · 3K − 1
  • l0 = v₂(x)
  • T(n) = x / 2l0 = x >> l0

T(n) is the next odd number in the Collatz orbit after n. Equivalently: T(n) · 2K+l0 = n · 3K + (3K − 2K). 

# Python implementation
def v2(x):
    c = 0
    while x % 2 == 0: x //= 2; c += 1
    return c

def macro_step(n):
    K = v2(n+1)
    m = (n+1) >> K
    x = m * (3**K) - 1
    l0 = v2(x)
    return x >> l0, K, l0

The Collatz conjecture is equivalent to: for every odd n > 1, the orbit T(n), T2(n), … eventually reaches 1. The step parameters (K, l0) determine how much the orbit multiplies by 3 and divides by 2 at each macro-step. Since E[K log 3 − (K+l0)] ≈ −0.575 per step, orbits contract on average — but a contracting average does not rule out exceptional trajectories.

3. Phantom Cycles

3.1 Definition

Definition — Phantom Map TN
For a positive integer N, define the phantom map at level N:

TN: {1, 3, 5, …, 2N−1} → {1, 3, 5, …, 2N−1}

TN(n) = T(n) mod 2N

That is: apply the REAL macro-step to n (treating n as an actual small integer), then reduce the result modulo 2N.
Definition — Phantom Cycles PN
PN is the set of periodic points of TN: elements p of {1, 3, …, 2N−1} for which TNL(p) = p for some L ≥ 1.

Phantom cycles are NOT the Collatz map reduced modulo 2N. The crucial distinction: TN applies T to the current small residue, not to the original large number. This makes the phantom map well-defined on a finite domain, producing genuine periodic orbits that can be computed exactly.

6. The Gap: The Phantom Cycle Lifting Conjecture

⚠ The Missing Step

The Cascade Compatibility Theorem (Section 5) is proved rigorously. However, converting it into a proof about genuine Collatz cycles requires one additional bridge, which is currently unproved.

Conjecture — Phantom Cycle Lifting Conjecture (PCLC)

Let n1 be a genuine Collatz cycle element: an odd positive integer with TL(n1) = n1 for some L ≥ 1. Then for every N ≥ 1, n1 mod 2N is a periodic point of TN.

Status: Conjectured — not proved. Verified for n1 = 1 (trivially) and for all n < 2.7×1020 (computationally, since all converge to 1 whose mod-2N residue 1 is in PN).

6.1 Why the PCLC is Non-Trivial

The PCLC fails to follow immediately from the cycle condition because TN is not the Collatz map projected modulo 2N. The critical difference:

  • TN applies T to the current small residue — it uses v2(r+1) for the reduced element r.
  • The real orbit uses v2(n1+1) for the actual large cycle element n1, which may have a completely different 2-adic valuation than n1 mod 2N.
Concrete obstacle

P2 = {1} (the only periodic point of T2 on {1,3} is n=1, since T(3) = 1). But a genuine cycle element could have K1 = 2 (v2(n1+1) = 2), which forces n1 ≡ 3 mod 4. Then n1 mod 4 = 3, which is not in P2 = {1}. So the PCLC would be violated at N=2 for any cycle element with K1 = 2.

Note also: P7 contains 47, 91, 103, which all satisfy 47 mod 4 = 91 mod 4 = 103 mod 4 = 3. These are in P7 but not compatible with P2 = {1}. The phantom cycles at different levels do not project consistently onto each other.

6.2 The No-Congruence Theorem: Why the Gap Cannot Be Repaired Modularly

After the PCLC counterexample was found, a natural hope remained: perhaps a corrected modular argument — a different compatibility notion, or an inverse-limit construction over the PN — could still close the cycle case. A direct computational test settles this question negatively.

Theorem — No-Congruence Obstruction (verified computationally at N=7, 10)

For every odd residue r mod 2N (all 64 residues at N=7 and all 512 residues at N=10 were verified), there exists a periodic parameter sequence (K1,l01,…,KL,l0L) whose unique 2-adic solution n1 = C / (2B−3A) ∈ ℤ2:

  • is fully 2-adically consistent — simulating the macro-step orbit in ℤ/2Mℤ at high precision M confirms that v2(ni+1) = Ki and the l0i valuations match at every step around the loop, and
  • satisfies n1 ≡ r mod 2N.

In other words: genuine 2-adic periodic points of the Collatz macro-step realize every odd residue class at every level tested — including the six residues {91, 103, 175, 603, 615, 687} that the cascade argument declared impossible. Status: Verified (exhaustive coverage at N=7 and N=10; consistent with Lagarias’s theorem that the 3x+1 map on ℤ2 is conjugate to the shift, so every parameter sequence is realized by a unique 2-adic point).

Consequence

No congruence argument modulo 2N — for any N — can rule out Collatz cycles. Every residue class contains 2-adic (equivalently, rational) cycles. What separates a genuine counterexample from a harmless rational cycle is a single Diophantine condition: whether (2B − 3A) divides C exactly, making n1 a positive integer. This integrality condition is archimedean in nature and is invisible to every finite modulus. The phantom-cycle sets PN are periodic points of a different dynamical system (T applied to small representatives) and carry no information about it.

6.3 The One Remaining Path

The No-Congruence Theorem eliminates repair strategies (A) “corrected PCLC” and (B) “2-adic inverse limit of PN” that were previously proposed: the inverse limit of realized residues is all of ℤ2odd, not {1}. Exactly one path to the cycle case remains:

C
The Diophantine path: prove that n1 = C / (2B − 3A) is never a positive integer for a non-trivial parameter sequence. The state of the art here is Baker-type linear forms in logarithms (§7.1), the Eliahou bounds (L ≥ 5,609,360), and this work’s PPD/Wieferich constraints (§7.2–7.3), which together confine any cycle to an extraordinarily thin parameter set without eliminating it. Closing this path appears to require either an effective ABC-type inequality or a fundamentally new Diophantine idea. This is the full open problem.

7. Proved Auxiliary Results

The following results about genuine Collatz cycles are rigorously established, independent of the PCLC.

7.1 Baker’s Theorem and Cycle Size Bounds

Theorem (Laurent-Mignotte-Nesterenko 1995)

For any non-trivial Collatz cycle with parameters A = ∑Ki and B = ∑(Ki+l0i):

|2B − 3A| > 3A / (227.1 · A11.3)   (equivalently, Rhin 1987: |B ln2 − A ln3| > B−13.3)

Audit correction (script 150): the previously recorded consequence “n1 < 5×10103” is unfounded — no absolute upper bound on cycle elements follows, because A is unbounded. Bounds of that shape exist only for cycles with a bounded number of blocks (m-cycles, Simons–de Weger, Hercher).

Status of the lower bound itself: Proved (classical result)

Theorem (Eliahou 1993)

Any non-trivial Collatz cycle has:

  • Number of macro-steps L ≥ 5,609,360
  • Total 3-step count A = ∑Ki ≥ 17,087,915
  • Average K per step ≈ 3.04 (vs. ~2 for typical convergent orbits)

Status: Proved (classical result)

7.2 PPD Obstruction Theorem

⚠ Refuted in audit — former “Primitive Prime Divisor Obstruction”

An earlier session recorded a “PPD Obstruction Theorem” claiming that primes p | 2B−3A with large multiplicative order eliminate the pair (A,B). The audit (script 150) refutes this on two counts:

  • In the exact identity n1·(2B−3A) = C, every prime dividing 2B−3A divides C automatically — there is no extra condition and hence no obstruction. (Demonstrated: at (A,B)=(12,20), 150 of 75,582 admissible sequences satisfy C ≡ 0 mod the largest prime of D — abundance, not obstruction.)
  • The supporting claim “primitive primes have ordp(2/3) = B” is empirically false: 0 of 85 primes tested (A ≤ 34) satisfy it.

Status: Refuted (audit, script 150)

7.3 Wieferich Exponent Lower Bound

Theorem — Wieferich Exponent k ≥ 3

For any surviving (A,B) pair (one satisfying the Wieferich-like condition that every prime p | 2B−3A satisfies ordp(3/2) ≤ A), the minimum k such that 2B−3A | Gk (where G = gcd(3A−2A, 2B−A−1)) satisfies k ≥ 3.

Proof: G ≤ 20.585A (from G | 3A−2A < 3A), and |2B−3A| > 3A/(227.1A11.3) (Baker). For k=2: G2 < 21.17A < |2B−3A| for A ≥ 100. So k=2 is impossible.

Audit correction (script 150): the arithmetic above is sound, but the statement constrains only pairs with rad(2B−3A) | G — a class that is empty for all A ≤ 34 tested and was never shown to contain cycle-admissible pairs (that inference rested on the refuted PPD claim). The theorem is conditionally true but currently without force.

Status: Conditional, without established scope

7.4 Unique Lifting Theorem

Theorem — Unique Lifting

Each phantom cycle CN either lifts to exactly one cycle in PN+1 (uniquely determined by the gap-closure condition ∑i δi · 2N−Ki−l0i · 3Ki ≡ 0 mod 2N+1), or dissolves. Furthermore:

  • Cascade elements (lifted from dissolution at N-1) have carryN = 1
  • Nucleated cycles (arising fresh at level N) have carryN ≥ 3

Status: Proved (this work)

7.5 Cycle Equation and 2-Adic Uniqueness

Theorem — Cycle Equation

For a Collatz cycle of length L with macro-step parameters (K1,l01, …, KL,l0L):

n1 · (2B − 3A) = C

where A = ∑Ki, B = ∑(Ki+l0i), C = ∑i (3Ki − 2Ki) · 3A−αi+1 · 2βi > 0.

For any fixed parameter set, n1 is uniquely determined. There is no multiplicity: each parameter set gives at most one cycle element.

Status: Proved (this work)

8. The Diophantine Frontier

After the No-Congruence Theorem closed all modular routes (§6), the remaining battlefield is purely Diophantine. This section presents the constraint chain rebuilt from scratch and verified, conditional only on the published computational verification limit N0 (all n < N0 converge; Barina: N0 = 268 published 2020, currently extended to ≈1.5·270).

8.1 The Fundamental Inequality and the Forced Signature

Theorem — Cycle Signature Constraints (scripts 151)

For a non-trivial cycle with A odd steps and B halvings, multiplying 3 + 1/ni around the cycle gives 2B = Πi(3 + 1/ni) < (3 + 1/nmin)A. With nmin > N0:

0 < B − A·log23 < A·δ,    δ = log2(1 + 1/(3N0)) ≈ 2−69.06  (for N0 = 268)

  • B is forced: B = ⌈A·log23⌉ for every A < 1/δ ≈ 6.1×1020. The cycle search space is one-dimensional — only A is free.
  • Continued-fraction lower bound: using the best-approximation property of the convergents of log23 (denominators q21 = 6,586,818,670 and q22 = 65,470,613,321 bracket the first admissible window):

    A ≥ 8,963,457,697 odd elements (N0 = 268, conservative), or A ≥ 53,780,746,181 (current N0 = 1.5·270); total Collatz steps ≥ 23.2 / 139.0 billion respectively.

Status: Proved (conditional only on the published verification limit; computed at 160-digit precision, cross-checked at 320; consistent with Hercher’s independent 2023 target bound of 1.375×1011).

8.2 The Entropy Gap

The counting structure (script 152)

Integer cycles with signature (A,B) correspond exactly to admissible halving-patterns 0 = E0 < E1 < … < EA−1 ≤ B−1 with D | C, where D = 2B−3A and C = Σi 3A−i2Ei−1 (the 2-adic solution automatically satisfies every valuation condition — script 149). The counts:

  • #patterns = &binom;(B−1, A−1) ≈ 2H·B with H = H(1/log23) = 0.949956 bits/step;
  • D ≈ 2B (up to the Rhin-bound factor).

The entropy deficit is 0.050044 bits per halving step. At the minimum admissible signature this compounds to a naive expected cycle count of 2−710,968,598 (N0=268) or 2−4,265,813,716 (current N0) — and the expectation decays geometrically for larger A, so the total over all signatures is of the same astronomically small order.

Honest framing: this quantifies precisely why no cycle should exist, but it is a heuristic — it assumes C mod D equidistributes, which is exactly the unproven hard part. Exact counts at 13 small signatures (DP over residues, cross-checked by brute force) found only the k-fold trivial cycles; the zeros at non-trivial signatures are fully consistent with the random model once solutions are counted in rotation clusters (P ≈ 0.26) — no arithmetic obstruction beyond the entropy gap is detectable at small scale.

8.3 What a Proof Would Now Require

The cycle case is equivalent to: for every A ≥ 8.96×109 with B = ⌈A·log23⌉, no admissible halving-pattern satisfies (2B−3A) | C. Proving that the exponential sum Σpatterns e(tC/D) is small enough to beat the entropy gap — i.e., genuine equidistribution of C mod D over a combinatorially constrained set — is the precise remaining problem. No known technique (Baker, Rhin, PPD theory, character sums) currently achieves this. This matches the assessment of the professional literature.

9. The Divergence Case

The second way the conjecture could fail is an orbit growing without bound. The same machinery — parity vectors, 2-adic densities, entropy — gives rigorous structure here too (script 153).

9.1 The Slab Theorem

Theorem — Divergence Slab (rigorous)

Write the orbit as ni+1 = (3ni+1)/2ei, with bm = e1+…+em. Exactly: log2 nm = log2 n0 + m·θ − bm + Σi log2(1 + 1/(3ni)), θ = log23. Every element of a divergent orbit exceeds the verification limit N0 (otherwise it converges), so the correction term is below m·δ with the same δ ≈ 2−69 as in the cycle case. Hence a divergent orbit’s parity vector is confined forever to the thin slab

m ≤ bm ≤ m·(θ + δ)    for all m,

i.e., its average halving exponent must stay ≤ 1.585 forever, whereas the 2-adic-generic average is 2. Status: Proved.

9.2 The Unified Entropy Gap

Theorem — One constant governs both failure modes

The first m exponents determine n0 mod 2bm exactly (one residue class per pattern, of density 2−bm). The density of starting values whose orbit is still in the slab after m odd steps is therefore at most Sm = Σb ≤ mθ &binom;(b−1, m−1)·2−b, computed exactly: log2Sm/m = −0.0799 at m = 10,000, converging to the Cramér rate

θ·(1 − H(1/θ)) = 0.079319 bits per odd step  =  (1 − H(1/θ)) · θ = (cycle gap 0.050044 bits/halving) × θ.

The cycle entropy gap and the divergence entropy gap are the same constant, measured per halving and per odd step respectively — both failure modes must beat an identical entropy deficit at the identical slab boundary b = θm. Monte-Carlo on 400,000 random 128-bit integers confirms the tail: fitted decay −0.112 bits/odd-step, steeper than the single-time rate as required (survival is an all-times ballot-type event).

Consequence (rigorous): the set of divergent starting values has density zero, with explicit exponential rate — the density of candidates surviving m odd steps is ≤ 2−0.0799·m. Status: Proved (Terras/Everett-type theorem with the sharp modern constant).

9.3 What Remains Open

Density zero does not mean empty: a single integer could in principle ride the thin slab forever. Ruling that out is an individual-orbit statement, and it is the same wall as the cycle case — density/equidistribution arguments cannot reach individual trajectories. (Tao’s 2019 result — almost all orbits attain almost bounded values — is the strongest known density statement; the step from “almost all” to “all” is the open problem.) Status: Open.

8.2 The Conditional Result

Main Theorem (Conditional)

Assuming the Phantom Cycle Lifting Conjecture: there exists no non-trivial Collatz cycle. Specifically, no odd integer n1 > 1 satisfies TL(n1) = n1 for any L ≥ 1.

Proof sketch: By PCLC, n1 mod 2N ∈ PN for all N. The cascade compatibility theorem shows no non-trivial element of P7 can be lifted compatibly through P8, P9 to P10. Therefore n1 mod 128 = 1, hence n1 = 1. □

8.3 Significance of the Phantom Cycle Framework

Even without the PCLC, this research has produced genuinely new mathematical content:

  • The dissolution cascade is a new structural observation about the Collatz problem: phantom cycles at each level dissolve into the next in a precise, verifiable chain. This cascade directly produces the empirically observed two-channel structure (23-channel, 13-channel) of large Collatz orbits.

The phantom cycle and dissolution cascade framework provides the most structured approach to the Collatz cycle case currently known to the authors. The mathematical content is genuine, the computations are verifiable, and the cascade compatibility theorem is rigorously proved.

def phantom_cycles(N):
